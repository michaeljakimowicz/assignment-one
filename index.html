Let's Mark-Up A Page

This is a page of text that you can use to test your knowledge and skills in HTML. Even though it contains some instructions, markup all the text using the correct elements for each portion of text. There's even an image for you to include in the page. For the present time, you should only write the HTML code. Don't attempt to apply any type of style to the page. It should look very, very basic with no added colours or alignments of text or images. Remember: that's what CSS is for.

Here are some tips that you could use to help you markup the text:

  • Does the page have a heading that describes the page as a whole? Then use an <h1> element for that text.
  • Is there is list? Is the order of the elements in the list important or can they be in any order? If order isn't important, use a <ul> element. If order is important use a <ol> element.
  • Are they special characters in the text? They could things such as ( ) { } & * %. They should be written using ASCII codes. You can see the codes for the characters in the link below.
  • The image that you include in the markup on the page should be in the same folder as your HTML file for now.
